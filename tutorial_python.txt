INTRODUÇÃO TUTORIAL DE PYTHON
	Hoje iremos instalar o python e suas livrarias dentro de sua máquina.

EXPLICANDO O QUE É PYTHON
	O Python é uma linguagem de programação que foi criada nos anos 90 para ser uma linguagem interpretada de scripts, imperativa, funcional, de tipagem dinâmica e forte e orientada a objetos. Ela é uma das linguagens mais usadas no mundo, equiparado com JavaScript e C.

PASSO A PASSO FAZER O DOWNLOAD
	1- Abra seu navegador e entre no site para o download do python usando o link "https://www.python.org/downloads/";
	2- Clique no botão escrito "Download Python 3.12.3" para abaixar o instalador do python.
	3- Espere o arquivo ser baixado de seu navegador
	4- Sucesso!

PASSO A PASSO DE INSTALAÇÃO
	1- Após o download, abra o instalador.
	2- No início, clique na caixinha do "Add python.exe to PATH", que fica bem na seção do rodapé do instalador.
	3- Clique em "Install Now"
	4- Aguarde a instalação
	5- Sucesso!

AMBIENTES VIRTUAIS DO PYTHON: O QUE É?
	É como se fosse uma cópia do Python em uma subpasta do seu projeto. É util para evitar que problemas atinga o diretório principal do python, e também evitar um possível conflito entre livrarias que não são compatives uma a outra.

COMO CRIAR UM VENV (AMBIENTE VIRTUAL)
	1- Abra o seu cmd no admnistrador.
	2- Navegue até o caminho de seu projeto usando cd /caminho/do/projeto.
	3- Digite o comando "python -m venv seu_nome_do_seu_venv".

ATIVAR UM VENV NO WINDOWS:
	Digite "seu_nome_do_seu_venv\Scripts\activate" para ativar o seu ambiente virtual.

ATIVAR UM VENV NO MACOS E LINUX:
	Digite "source seu_nome_do_seu_venv\bin\activate" para ativar o seu ambiente virtual.

INTRODUÇÃO A INSTALAÇÃO DO VSCODE
	Hoje iremos mostrar o que é o VScode e a instalação dele na sua máquina.

EXPLICAÇÃO SOBRE O QUE É O VSCODE
	O Visual Studio Code (VScode) é uma ferramenta editora de código-fonte criada pela Microsoft e visa editar praticamente que quase todos as linguagens de código disponíveis no mercado. Possui o controle de versionamento do Git incorporado, realce de sintaxe, complemento inteligente de código, extensões para melhorar a sua edição do seu código, entre vários tipos de disponibilidades. É gratuito, de código aberto, e multiplataforma, sendo suportado para Windows, Linux e MacOS. É muito usado por vários desenvolvedores profissionais no mundo.

DOWNLOAD DO VSCODE:
	1- Abra o seu navegador.
	2- Na barra de endereço do link, digite: "https://code.visualstudio.com/"
	3- Clique no botão "Download for Windows", para caso for o sistema operacional for Windows, mas tem suporte para MacOS e Linux.
	4- Aguarde o arquivo de instalação do VScode ser baixado do seu navegador.
	5- Sucesso!

INSTALAÇÃO DO VSCODE:
	1- O instalador estará com o nome de "VSCodeUserSetup-x64-1.88.1". Ache-o e execute-o.
	2- Aceite os temos do contrato e clique em Próximo.
	3- Marque a opção de "Criar um atalho na área de trabalho"
	4- Se estiver desmarcado as duas opções de "Abrir um Code" no Windows Explorer, ative-os. Caso contrário, clique em próximo.
	5- Clique em Instalar.
	6- Aguarde a instalação.
	7- Após o processo, clique em Concluir.
	8- Sucesso!
